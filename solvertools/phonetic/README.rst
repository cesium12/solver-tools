.. _phonetic:

The `solvertools.phonetic` package
==================================

This package is for working with phonetic data, particularly
the pronunciation dictionary :ref:`phonetic.txt`.

Currently it contains one module, used in building `phonetic.txt` from the
format given in the CMU wordlist.
